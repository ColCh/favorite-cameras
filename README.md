## Launch

Share `build` directory and open `index.html`

1. `yarn global add pushstate-server` - static server
2. `yarn run serve` - serve and open page


## Development
`yarn start` 


## Tests
`yarn test` - uses `jest` for tests


## Сборка
`yarn run build` - builds files into `build`
